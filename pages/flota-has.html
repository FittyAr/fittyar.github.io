Gestión de Flotas

Flota-HAS

Aplicación local-first para la gestión integral de vehículos, choferes, mantenimientos, recaudaciones y balances individuales de ride-hailing. Desarrollada con .NET MAUI.

Ver código en GitHub Otros proyectos
Stack del Proyecto

Tecnologías Usadas

Framework .NET MAUI (Multi-platform App UI)
Persistencia SQLite & Entity Framework Core
Soporte OS Windows, Android, macOS, iOS

Características Principales

Diseñada para operar eficientemente de forma autónoma con una experiencia fluida y rápida.

Arquitectura Local-First

Usa SQLite localmente con Entity Framework Core para garantizar que todos los datos se almacenen de forma segura y estén disponibles de inmediato sin necesidad de internet.

Cero Fricción

Acceso directo a la aplicación y al panel de control de inmediato, sin flujos de registro complejos ni requerir inicios de sesión obligatorios.

Exportación de Datos

Permite exportar rápidamente todos los reportes operativos y financieros en formatos estándar como hojas de cálculo de Excel (.xlsx) y JSON.

Modelo de Dominio

Entidades que componen la lógica de negocio y operativa del sistema.

Vehículos

Control completo de marca, modelo, año, patente, kilometraje, seguro, estado del vehículo (activo, mantenimiento, fuera de servicio) y alertas de vencimiento de inspecciones técnicas (VTV/ITV).

Choferes

Registro exhaustivo de datos personales, licencia de conducir (categoría y vencimiento), estado (activo, suspendido, inactivo) y la asignación del vehículo correspondiente.

Mantenimientos

Gestión e historial de servicios preventivos y correctivos, costos de taller, insumos y kilometraje al momento de realizar la reparación.

Acuerdos Comerciales

Definición de los términos y montos comerciales pactados individualmente con cada chofer, estableciendo la frecuencia de liquidación (semanal, quincenal, mensual).

Recaudaciones

Carga diaria o periódica de los cobros resultantes de la operación de ride-hailing, detallando la fecha, el monto neto y el método de pago utilizado.

Movimientos Contables

Libro contable unificado de ingresos y egresos vinculados directamente a un vehículo o chofer, con asignación explícita del responsable del gasto.

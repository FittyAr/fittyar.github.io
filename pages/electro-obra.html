Gestión Operativa

ElectroObra

Sistema multiplataforma para el control de flujo de caja y la gestión diaria de obras y operarios en pymes de mantenimiento y construcción. Desarrollado con Avalonia UI y .NET 10.

Stack del Proyecto

Tecnologías Usadas

Plataforma .NET 10.0 SDK & C#
Interfaz UI Avalonia UI 12.0+ (MVVM)
Persistencia SQLite & Entity Framework Core

Filosofía del Proyecto

Enfoque centrado en la realidad operativa cotidiana de las empresas del sector.

Flujo de Caja Real

Monitorea los movimientos reales de dinero (efectivo, bancos, monotributo, seguros, materiales y herramientas) en lugar de una contabilidad fiscal compleja.

Flexibilidad en el Mundo Real

Diseñado contemplando inasistencias injustificadas, adelantos de sueldo a empleados, pagos parciales de clientes y saldos pendientes de cobro.

Diseño Premium y Portable

Interfaz fluida e interactiva con Avalonia UI, preparada para compilarse nativamente para Escritorio (Windows, Linux, macOS), Móvil (Android, iOS) y Web.

Arquitectura de Software

Desarrollado bajo los principios de Clean Architecture para garantizar mantenibilidad a largo plazo y testabilidad.

Core (Domain)

La capa central del negocio. Contiene las entidades puras de dominio (Movement, Client, Job, Employee), enumeraciones, especificaciones y las interfaces de repositorio.

Application

Implementación de los casos de uso, DTOs (Data Transfer Objects), validación de negocio mediante FluentValidation y mapeo de objetos simplificado a través de Mapster.

Infrastructure

Maneja la persistencia en base de datos usando Entity Framework Core sobre SQLite, la escritura del sistema de archivos, exportación de reportes a PDF/Excel y el logging.

UI Layer (Avalonia)

Interfaz de usuario declarativa multiplataforma estructurada bajo el patrón MVVM, permitiendo compartir el 100% de la lógica visual entre sistemas operativos.

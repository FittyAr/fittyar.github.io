Software de Gestión Empresarial

Una aplicación de escritorio de alto rendimiento para la administración corporativa y el monitoreo de recursos locales en tiempo real, desarrollada bajo un esquema de código privado.

Caso de Estudio & Contexto Técnico

Este sistema fue construido a medida para resolver la gestión integrada de inventarios, facturación, auditoría y el monitoreo de servicios locales en una infraestructura física distribuida. Debido a acuerdos de confidencialidad (NDA), el repositorio y los detalles del cliente permanecen privados, pero su arquitectura y métricas de desarrollo ilustran un gran ejercicio de ingeniería.

Arquitectura de Software

El software está organizado como un **Monolito Modular** estructurado bajo principios de **Clean Architecture**:

  • Frontend de Escritorio (Avalonia UI): Una GUI fluida escrita en XAML y C# que se compila y ejecuta de manera nativa tanto en entornos corporativos Windows como en estaciones de trabajo Linux, con renderizado acelerado por hardware para animaciones sin latencia.
  • Desacoplamiento de Negocio: Separación rigurosa de las capas de Aplicación, Dominio e Infraestructura, comunicadas mediante el envío de comandos/consultas (CQRS).
  • Módulo de Monitoreo Local: Un demonio colector secundario (Linux Daemon / Systemd Service) escrito en .NET Core que se encarga de auditar en tiempo real el uso de RAM, CPU, espacio en disco y el estado de los contenedores Docker en los nodos locales.
  • Base de Datos y Persistencia: Diseñado con soporte para múltiples motores (usando Entity Framework Core), operando con transaccionalidad robusta y políticas de copias de seguridad automatizadas.

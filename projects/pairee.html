Pairee

Un gestor de archivos de doble panel para terminal rápido, modular y completamente multiplataforma (compatible con Linux, Windows y macOS), desarrollado en Rust.

Modernizando la Navegación TUI

Inspirado en clásicos indiscutibles como Norton Commander y Far Manager, Pairee moderniza la experiencia del terminal file manager. Está diseñado para ofrecer una navegación ágil y fluida basada enteramente en teclado, siendo ideal tanto para desarrolladores como para administradores de sistemas en cualquier sistema operativo.

Arquitectura y Características Técnicas

El motor de Pairee explota las bondades de seguridad y concurrencia de Rust:

  • Concurrencia Asíncrona (Tokio): Las operaciones pesadas de sistema de archivos (copiar, mover, eliminar, wipear) se ejecutan en hilos de fondo asíncronos para evitar bloquear la interfaz de usuario.
  • Seguimiento de Tareas: Permite visualizar el progreso de operaciones simultáneas con barras de estado dinámicas y cancelar procesos concurrentes directamente desde la TUI.
  • Atajos de Teclado Flexibles: Soporte para múltiples esquemas de control (distribución clásica F1-F10, navegación Vim y atajos modernos).
  • Temas de Color: Personalización de paletas visuales (Slate, Blue, etc.) adaptadas a la estética del emulador de terminal.
  • Herramientas Integradas: Búsqueda avanzada de archivos por nombre o contenido, comparador de directorios, menú personalizado de comandos rápidos y administrador de procesos del OS.
